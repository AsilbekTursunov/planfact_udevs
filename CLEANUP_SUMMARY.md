## 🔄 Что изменилось для разработчиков

### Раньше (удалено)
```javascript
// Запрос шел через Next.js API route (прокси)
Client → Next.js API Route → U-code API
         (/api/operations)

// Код
const response = await fetch('/api/operations/create', {
  method: 'POST',
  body: JSON.stringify(data)
})
```

### Теперь
```javascript
// Прямой запрос к U-code API
Client → U-code API

// Код
import { operationsAPI } from '@/lib/api/ucode/operations'
const response = await operationsAPI.create(data)
```

## 📝 Обратная совместимость

`lib/api/dashboard.js` сохранен для обратной совместимости:

```javascript
// Старый код продолжит работать
import { dashboardAPI } from '@/lib/api/dashboard'
const data = await dashboardAPI.getOperations(params)

// Но рекомендуется использовать новый
import { operationsAPI } from '@/lib/api/ucode/operations'
const data = await operationsAPI.getList(params)
```

Все методы в `dashboardAPI` помечены как `@deprecated` и выводят предупреждения в консоль.
